# 多功能轉換工具

這是一個集成了多種實用功能的網頁工具，無需安裝任何軟體，直接在瀏覽器中使用。

## 功能介紹

### 1. 文字轉換
- **功能**：將 Unicode 或 JSON 格式的文字轉換為正常的中文
- **支援格式**：
  - Unicode：`\uXXXX`、`\u{XXXX}`
  - JSON 字串轉換
- **使用場景**：處理從API回傳的編碼文字、JSON資料中的中文

### 2. 圖片轉 Base64
- **功能**：將圖片檔案轉換為 Base64 編碼字串
- **支援格式**：所有常見圖片格式（PNG, JPG, GIF, WebP 等）
- **操作方式**：拖曳檔案或點擊選擇
- **使用場景**：網頁開發中嵌入圖片、API資料傳輸

### 3. Placeholder 圖片生成
- **功能**：快速生成佔位圖片
- **自訂選項**：
  - 寬度、高度
  - 背景色、文字色
  - 文字內容
  - 檔案格式（PNG, JPG, WebP）
- **使用場景**：網頁開發時的臨時圖片、設計稿製作

### 4. QR Code 生成
- **功能**：將文字或網址轉換為QR碼
- **自訂選項**：
  - 尺寸（256x256, 512x512, 1024x1024）
  - 容錯等級（L/M/Q/H）
- **操作**：支援下載和複製Base64
- **使用場景**：製作名片、分享連結、行動應用

### 5. SVG 轉 PNG
- **功能**：將 SVG 向量圖轉換為 PNG 點陣圖
- **操作方式**：拖曳 SVG 檔案或點擊選擇
- **輸出**：PNG 圖片檔案和 Base64 字串
- **使用場景**：圖片格式轉換、網頁相容性處理

## 使用說明

1. 開啟 `index.html` 檔案
2. 點擊上方分頁切換功能
3. 根據需求輸入內容或上傳檔案
4. 點擊對應按鈕執行轉換
5. 複製結果或下載檔案

## 檔案說明

- `index.html`：主要工具檔案，包含所有功能
- `qrcode.min.js`：QR Code 生成函式庫
- `.nojekyll`：停用 Jekyll，讓 GitHub Pages 直接提供靜態檔案

## 注意事項

- **圖示素材**：本專案未提供圖示檔案，如需圖示請自行設計製作

## 特色

- 🌐 **純前端**：無需後端服務，資料不會上傳到伺服器
- 🛠️ **多功能**：五種實用工具整合在一個頁面
- 🎨 **直覺介面**：清晰的分頁設計，操作簡單
- ⚡ **即時轉換**：快速處理，即時預覽結果
